# Консольный калькулятор на C++

Простой калькулятор с сохранением истории операций в файл.

## Функции
- Сложение, вычитание, умножение, деление
- Сохранение всех вычислений в `CalculationOperations.txt`
- Защита от деления на ноль
- Русский интерфейс

## Сборка
Компилируется любым современным компилятором C++ (g++, MSVC):
```bash
g++ -o calc Calculator.cpp
